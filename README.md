### Portfolio Website

A modern, fully responsive personal portfolio website designed to showcase my technical skills, projects, certifications, resume, and professional achievements. The website features an intuitive user interface, smooth animations, and optimized performance to provide an engaging user experience across all devices. It serves as a central platform for recruiters and collaborators to explore my work, download my resume, and connect with me through integrated contact and social media links.

**Key Features:**

* Responsive design for desktop, tablet, and mobile devices
* Interactive project showcase with live demos and GitHub links
* Skills, education, certifications, and experience sections
* Resume download functionality
* Contact form with social media integration
* Modern UI/UX with smooth animations and fast loading speed

**Technologies Used:** HTML5, CSS3, JavaScript, Bootstrap, Font Awesome, Responsive Web Design, GitHub Pages.
